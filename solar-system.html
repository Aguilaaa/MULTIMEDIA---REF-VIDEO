Project details

Built in Blender 3.3 with Python in the Scripting workspace. High-resolution textures from Solar System Scope. Script assets and notes are organized for Sun and each planet.

  • Render: ~13.5 hours; 1920×1080 @ 24 fps; 3500 frames (~2 min 25 s).
  • Pipeline: Cycles, GPU-focused settings, camera tour from Sun through planets, zoom-out system view.
  • Python scripts: Sun and planets (Google Drive)
